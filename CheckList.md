# Функціонал програми для гостя (не авторизований): 
- Авторизація / Реєстрація + 
- Перегляд каталогу медіаконтенту 
- Пошук контенту
# Функціонал програми для авторизованого користувача: 
- Перегляд каталогу медіаконтенту (потім) 
- Пошук контенту (потім) 
- Додавання об'єкту до "Вибраного" + 
- Додавання статусу об'єкту (Переглянуто / У планах | додатково для серіалів або аніме Дивлюся / На паузі / Занедбано) + 
- створення кастомних колекцій. Тобто, користувач може занести контент під свої категорії. +- 
Наприклад якусь комедію можна додати до створеної колекції "Для веселого вечора". 
- Додавання рецензії на медіаконтент. Оцінка (обов'язково) та Відгук (опціонально). + 
- (Під питанням) Якщо об'єкта немає в каталозі, користувач створює свій (буде тільки назва і рік релізу).
# Що потрібно від програми: 
- Створення персональних рекомендацій на основі рецензій користувача (алгоритм). - 
- Можливість пошуку при невдалому формулюванні (ІІ). - 
Наприклад: Хочу щось атмосферне чи мотиваційне на вечір. Користувачеві видає список фільмів, 
відповідних під опис (наприклад MoneyBall, The Pursuit of Happyness, Good Will Hunting) 
- Посилання на Netflix, Megogo чи Crunchroll, де можна подивитися
